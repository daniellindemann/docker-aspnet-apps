# Docker for ASP.NET apps

This project shows how to setup Docker containers for ASP.NET applications using best practices and further optimization concepts. 

## Prerequisites

- [Docker](https://www.docker.com/get-started/) or similar container runtime
- [.NET 10](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)

## Setup

### Option 1

Everything runs in a dev container. Open it with Visual Studio Code. You can now execute all scripts and tools.

> 💡 [Developing inside a Container](https://code.visualstudio.com/docs/devcontainers/containers)

### Option 2

- Install Tools
    - Dockle (https://github.com/goodwithtech/dockle)
    - Hadolint (https://github.com/hadolint/hadolint)
    - Trivy (https://github.com/aquasecurity/trivy)
    - Copacetic (https://github.com/project-copacetic/copacetic)
    - Notation (https://github.com/notaryproject/notation)
    - Notation AKV Plugin (https://github.com/Azure/notation-azure-kv)
